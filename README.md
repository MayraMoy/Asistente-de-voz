# Reconocimiento de Voz para Búsquedas en MercadoLibre

<p>
  Aplicación en Python que permite realizar búsquedas en MercadoLibre mediante comandos de voz, integrando reconocimiento de audio, síntesis de voz y automatización web.
</p>

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Status](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-green)

## Tech Stack
| Tecnología            | Uso                          |
| --------------------- | ---------------------------- |
| Python                | Lógica principal             |
| speech_recognition    | Reconocimiento de voz        |
| pyttsx3               | Síntesis de voz              |
| webbrowser            | Automatización de navegación |

## Funcionalidades

Captura de voz en tiempo real
Interpretación de comandos hablados
Respuesta auditiva del sistema
Búsqueda automática en MercadoLibre

Ejemplo de uso:

Usuario: "Buscar notebook gamer"
Sistema: Abre MercadoLibre con resultados

## Requisitos
✔️ Python 3.x
✔️ Micrófono funcional
✔️ Conexión a internet
